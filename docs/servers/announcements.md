## Installation

```bash
npm install -g @access-mcp/announcements
```

Add to your Claude Desktop configuration:

```json
{
  "mcpServers": {
    "announcements": {
      "command": "npx",
      "args": ["@access-mcp/announcements"]
    }
  }
}
```
